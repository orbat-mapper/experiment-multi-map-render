# Multi map rendering

An ongoing experiment on how to render map data through a common interface that supports multiple map libraries.

## Getting started

Requisites:

- Node.js v22+
- pnpm package manager

Clone repo:

    $ git clone https://github.com/orbat-mapper/experiment-multi-map-render.git

Navigate to project root:

    $ cd experiment-multi-map-render

Install dependencies:

    $ pnpm install

To start a development server run:

    $ pnpm dev

The demo is now running on http://localhost:5173/.
